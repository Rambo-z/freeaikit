## Description
FreeAIKit is a collection of 57+ free online tools that run entirely in your browser. Image compression, PDF merge, background removal, OCR, and more — all powered by WebAssembly. No signup, no upload to server, no watermark.

**What makes it different:**
- 🔒 Privacy-first: Your files are processed locally, never uploaded
- ⚡ No signup required: Just open and use
- 🌍 Available in English, Spanish, and Portuguese
- 💰 100% free with no limits

**Popular tools:**
- AI Background Remover (ONNX AI model, runs in browser)
- Image Compressor (MozJPEG/libwebp/libimagequant — TinyPNG-level compression)
- PDF Merge & Split
- Image to Text (OCR with 100+ languages)
- 50+ more tools for images, PDFs, text, dev tools, and SEO

**Tech stack:** Next.js 16, React 19, WebAssembly, Cloudflare Pages
